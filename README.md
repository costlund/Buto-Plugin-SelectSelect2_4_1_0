# Buto-Plugin-SelectSelect2_4_1_0
Transform selectbox to searchable selectbox.

## Include
```
type: widget
data:
  plugin: 'select/select2_4_1_0'
  method: include
```

## Usage
```
type: script
innerHTML: |
  $(document).ready(function() {
      $('#my_form_coworker').select2();
  });
```
